Filtros en templates


ADD

En este ejercicio tenemos la siguiente lista: 'value':[1,2,3]

Lo que hacemos con el filtro "add" en este caso es sumar 2 a cada uno de los valores de la lista, que en este caso son [1,2,3], y el resultado que nos arroja es:


    {% for add_numero in value %}

  • {{add_numero|add:"2"}}
    • {% endfor %}


En este ejercicio tenemos la siguiente lista: 'lista':["Nicolas P.", "Juan Carlo P.", "Sara V.", "Maria Alejandra R."]

Lo que hacemos con el filtro "add" en este caso es agregarle el nombre de "Andres Barragan" a cada uno de los nombres que se encuentran en la lista:


    {% for add_nombre in lista %}

  • {{add_nombre|add:" (Andres Barragan)"}}
    • {% endfor %}


CUT

En este caso al momento de usar el filtro "cut" estamos eliminando de la cadena de texto la cual es:

'correoE': "nicolas. paulo. vega. 06 @ gmail. com"

La cantidad de espacios que existan en la cadena anterior mostrando el siguiente resultado:

  • {{correoE|cut:" "}}



DIVISIBLEBY

En este caso, se utiliza el filtro "divisibleby" para marcar si el número que nosotros escojamos es divisible por el número que se encuentra en una variable. Por ejemplo, tenemos lo siguiente:

'divisible': 30

Si tenemos que el número 30 es divisible por 5, entonces el resultado que nos debe salir es True. Pero si no es divisible por tal número, debería salir False, como lo siguiente:

  • Es divisible por 5? ({{divisible|divisibleby:"5"}})
  • Es divisible por 7? ({{divisible|divisibleby:"7"}})
  • Es divisible por 2? ({{divisible|divisibleby:"2"}})
  • Es divisible por 9? ({{divisible|divisibleby:"9"}})



JOIN

En este caso usamos el filtro "join" para unir una línea de texto a una lista ya creada. Un ejemplo puede ser con la lista de nombres:

'lista': ["Nicolas P.", "Juan Carlo P.", "Sara V.", "Maria Alejandra R."]

Podemos añadirle al texto una medida de separación que no sean las comas; pueden ser un doble slash (//). El resultado será el siguiente:

  • {{lista|join:" // "}}



LENGTH

En este caso usamos el filtro "length" para calcular la longitud de una cadena de texto y una lista, las cuales son: (Cadena de texto: 'letras': "Hola mundo!!") y (Lista: 'value': [1,2,3]). El resultado de cada uno debería ser el siguiente:

  • Cadena de texto: {{letras|length}}
  • Lista: {{value|length}}



LENGTH_IS

En este caso, vamos a calcular la longitud de una cadena de texto y una lista, las cuales son: (Cadena de texto: 'letras': "Hola mundo!!") y (Lista: 'value': [1,2,3]). Pero en esta ocasión, vamos a establecer que el filtro "length" cumpla una condición y nos muestre el resultado como True. Sin embargo, si no se cumple, el resultado sería False. Para este ejemplo, vamos a decir que para la cadena de texto la longitud debe ser de 12 y en la lista la longitud debería ser de 5. Veamos el resultado:

  • Cadena de texto: {{letras|length_is:"12"}}
  • Lista: {{value|length_is:"5"}}



RANDOM

Para este caso, usamos el filtro "random" cuando queremos mostrar un elemento aleatorio. Un ejemplo puede ser de una lista de texto ('lista': ["Nicolas P.", "Juan Carlo P.", "Sara V.", "Maria Alejandra R."]) y una lista numérica ('value': [1, 2, 3]). El resultado sería el siguiente:

  • lista de texto: {{lista|random}}
  • Lista numérica: {{value|random}}



SLICE

En este caso, lo que vamos a hacer es utilizar el filtro "slice" para devolver una porción de una lista. Para este caso, utilizaremos una lista la cual es la siguiente:

'aprendices': ["Juan Diego", "Mathew Guarnizo", "Maria Fernanda", "Nicolas Paulo", "Miguel Angel", "Maicol Esteban", "Kevin Londoño", "Marlon", "Maria Jose", "Nataly"]

Ahora vamos a escoger una porción de 5 elementos de la lista y el resultado debería ser el siguiente:

  • {{aprendices|slice:"4"}}



SLUGIFY

En este caso, lo que hacemos con el filtro "slugify" es reemplazar los espacios con guiones. Entonces, si tenemos el siguiente texto: 'correoE': "nicolas. paulo. vega. 06 @ gmail. com", el texto ya no tendrá espacios, sino que tendríamos lo siguiente:

  • texto: {{correoE|slugify}}
